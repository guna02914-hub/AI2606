# Infinite Interval Timer

HTML/CSS/JavaScript로 만든 무한 반복 인터벌 타이머입니다.

## 주요 기능

- 시간·분·초 단위 직접 입력
- ON/OFF 버튼으로 타이머 시작 및 정지
- 타이머 종료 후 자동 재시작
- OFF를 누르기 전까지 무한 반복
- 반복 횟수 표시
- 사운드 볼륨 조절
- 사운드 테스트 버튼
- 입력한 문장 읽어주기 기능
- 타이머 종료 시 입력한 문장 자동 읽기
- 문장 읽기 테스트 버튼
- 브라우저 localStorage 저장
- 별도 mp3 없이 Web Audio API로 알림음 재생
- Web Speech API로 문장 음성 출력

## 사용 방법

1. 시간, 분, 초 입력칸에 원하는 간격을 입력합니다.
2. 읽어줄 문장 입력칸에 원하는 문장을 입력합니다.
3. 사운드 볼륨을 조절합니다.
4. 문장 읽기 테스트 버튼으로 음성이 나오는지 확인합니다.
5. ON 버튼을 누르면 타이머가 시작됩니다.
6. 타이머가 끝나면 알림음이 울리고 입력한 문장을 읽은 뒤 자동으로 다시 시작됩니다.
7. OFF 버튼을 누르면 무한 반복이 멈춥니다.

## GitHub Pages 배포 방법

1. GitHub Repository의 Settings로 이동합니다.
2. Pages 메뉴를 엽니다.
3. Source를 `Deploy from a branch`로 설정합니다.
4. Branch를 `main`, Folder를 `/root`로 선택합니다.
5. Save를 누르면 잠시 후 배포 주소가 생성됩니다.

## 주의사항

웹 타이머는 브라우저 탭이 열려 있을 때 가장 안정적으로 작동합니다. 브라우저 정책상 알림음과 문장 읽기는 사용자가 ON 버튼, 사운드 테스트 버튼, 문장 읽기 테스트 버튼을 직접 누른 뒤 정상적으로 재생됩니다.
